# Snake Game

## Description

A classic Snake Game developed using HTML, CSS, and JavaScript. Control the snake, eat food, and try to achieve the highest possible score without colliding with the walls or yourself.

## Features

* Keyboard controls
* Random food generation
* Score tracking
* Game Over detection
* Smooth gameplay

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the project.
2. Open `index.html` in a web browser.
3. Use the arrow keys to play.
