Frequently Asked Questions ❓

Quick answers to common questions.

Is Spendly free?
Yes. It is completely free and runs locally in your browser.
Where is my data stored?
All data is stored in your browser using localStorage. No server is involved.
Can I use it offline?
Yes. Since it uses localStorage, it works fully offline.
Can I delete my data?
Yes. You can clear localStorage from browser settings.
