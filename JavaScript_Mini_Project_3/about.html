Spendly helps you track, understand, and improve your financial habits.

📌 What is Spendly?

Spendly is a lightweight personal expense tracker built using pure frontend technologies. It allows users to manage income, expenses, and reports — all stored locally in the browser.

🚀 Features

📊 Track Transactions

Add income & expenses with categories.

📅 Reports

Monthly & yearly financial insights.

🗂 Categories

Understand where money is spent.

🔒 Private

100% offline using localStorage.

⚙️ Technology

  • HTML – Structure
  • CSS – Styling
  • JavaScript – Logic
  • LocalStorage – Data storage

🎯 Purpose

This project demonstrates frontend development skills including UI design, data handling, and building a complete application without backend.
